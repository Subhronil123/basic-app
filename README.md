# React Analytics Dashboard
A modern, production-style analytics dashboard built with React 18 + Vite, featuring real-world UI patterns like analytics filters, KPI summaries, charts, data tables, theming, and skeleton loaders.
This project focuses on clean architecture, UX polish, and scalability, rather than backend complexity.
## Features
### Core Dashboard
KPI summary cards
User growth & performance charts
Responsive layout (desktop-first)
Clean visual hierarchy
### Advanced Analytics
Metric switching (Users / Revenue)
Date range filtering
Preset date ranges (Today, Last 7, Last 30, This Month)
KPI summary auto-calculated from filtered data
Comparison mode (current vs previous)
### Data Table
Client-side search
Column sorting
Pagination
Reusable table component
### UI & UX
Dark / Light theme toggle
Persistent user preferences
Skeleton loaders for smooth loading states
Accessible, minimal Tailwind styling
### Settings Page
Theme control
Compact mode
Profile section (mock)
Reset preferences
## Tech Stack
Frontend: React 18, Vite
Styling: Tailwind CSS
Charts: Recharts
Routing: react-router-dom v6 (createBrowserRouter)
State Management: React Context + Hooks
Data: Mock JSON (backend-ready)
